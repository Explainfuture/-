这是一个简单的学生管理系统
主要功能如下
- 能对学生成绩进行录入和删除；
- 能浏览所有学生的成绩；
- 能按学号或姓名查询某学生成绩信息；
- 统计与排序：根据成绩排序；统计某一分数段的学生人数。
- 结果格式化显示。
